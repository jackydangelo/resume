## Personal resume
Based on [Jake Gutierrez](https://github.com/jakegut)'s work. I found this template surfing on reddit, i don't know who Jake is.<br>
I modified the template introducing geometry class for code more robust.<br>
Then I modified some things for being more ATS-Friedly.<br>
Last I tested GitHub pages and GitHub actions, just for fun.<br> 
### Quick use
Get started quickly using Overleaf and this code as template.<br>
The code is optimised for CVs in Italian. To change the language, search for the word “Italian” and replace it with your language.
